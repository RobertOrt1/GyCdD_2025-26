# Proyecto 6 - Evaluación de Madurez con UNE 0080

**Empresa:** EnergiTech | **Asignatura:** Gobierno y Gestión del Dato - MUBDCN, UCLM
**Marco de referencia:** UNE 0080 - MAMD (basado en ISO/IEC 33000) | **Sesión:** 14

## Contexto

Los cinco proyectos anteriores han generado los componentes base del gobierno del dato en EnergiTech: el proceso de negocio modelado en BPMN (P1), los repositorios de metadatos en OpenMetadata (P2), el repositorio de datos maestros con DDL ejecutado en Spartan (P3), el modelo de calidad con seis medidas ejecutables (P4) y los procedimientos de control con cuadro de mandos y no conformidades (P5).

Este proyecto aplica el modelo de madurez MAMD de UNE 0080 para determinar en qué nivel se encuentra EnergiTech con todo ese trabajo realizado, identificar las brechas que impiden subir al siguiente nivel y definir un plan de mejora concreto.

## Tareas

**Tarea 1 - Autoevaluación del nivel de madurez**
Evaluar cada proceso implementado (ProcDat, MetDat, MDM, PlanDQ, CtrlDQ) con la escala F/L/P/N de UNE 0080. Derivar el nivel de madurez actual de la organización y justificarlo con las evidencias generadas en P1 a P5.

**Tarea 2 - Plan de mejora**
Definir las acciones necesarias para alcanzar el Nivel 1 (Realizado) y, a continuación, el Nivel 2 (Gestionado). Asignar responsables y plazos a cada acción.
